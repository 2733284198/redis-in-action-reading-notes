# Redis的Strings类型

Redis的STRINGs类型就像其他编程语言或其他键值存储类型一样。

![图1.1](images/1.2.1-1.png)


STRINGs支持的操作方式和其他键值对存储的操作方式一样。我们可以``GET`` valeus, ``SET`` values, ``DEL`` values.

## 表 1.3 STRING类型可用的命令

| 命令 | 作用 |
| -- | -- |
| GET | 获取给定key存储的数据 |
| SET | 设置给定key存储的数据 |
| DEL | 删除给定key存储的数据(所有类型适用) |
